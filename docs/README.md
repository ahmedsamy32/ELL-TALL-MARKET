## 🗄️ قاعدة البيانات

### الجداول الرئيسية

1. **profiles** - معلومات المستخدمين
2. **stores** - المتاجر
3. **products** - المنتجات
4. **categories** - الفئات
5. **orders** - الطلبات
6. **order_items** - عناصر الطلبات
7. **cart_items** - سلة التسوق
8. **addresses** - العناوين
9. **favorites** - المفضلة
10. **reviews** - التقييمات

**للمزيد:** راجع `../supabase/migrations/Supabase_schema.sql`
