# 🔐 دليل تحديد نوع المستخدم أثناء التسجيل

## 🎯 **السؤال: كيف أحدد إذا كان المستخدم أدمن أم لا أثناء التسجيل؟**

### ✅ **الطرق المتاحة:**

#### 1. **بالإيميل المحدد مسبقاً** (الأسهل):
```dart
// الإيميلات المسموحة للأدمن:
admin@elltall.com
admin@admin.com  
super@elltall.com
root@elltall.com
```

#### 2. **بالدومين المخصص**:
```dart
// أي إيميل ينتهي بـ:
@elltall.com
@admin.elltall.com
```

#### 3. **بالكود السري**:
```dart
// الكود السري: ADMIN_SECRET_2024
// يُدخل في حقل خاص أثناء التسجيل
```

#### 4. **بكلمة مرور خاصة**:
```dart
// كلمات مرور مخصصة للأدمن:
ADMIN_MASTER_2024
SUPER_ADMIN_KEY
```

### 🛠 **كيفية الاستخدام:**

#### أ) **للمطور - إعداد الإعدادات:**
```dart
// في AdminDetectionConfig
static const List<String> adminEmails = [
  'admin@elltall.com',     // إيميل افتراضي
  'myemail@elltall.com',   // أضف إيميلك هنا
];

static const String adminSecretCode = 'MY_SECRET_2024'; // غير الكود
```

#### ب) **للمستخدم - طرق التسجيل كأدمن:**

##### الطريقة 1: بالإيميل
1. اختر إيميل من القائمة المسموحة
2. سجل كالعادة
3. ✅ سيصبح أدمن تلقائياً

##### الطريقة 2: بالكود السري
1. استخدم أي إيميل عادي
2. فعّل خيار "تسجيل كأدمن" 
3. أدخل الكود السري
4. ✅ سيصبح أدمن

##### الطريقة 3: بكلمة مرور خاصة
1. استخدم أي إيميل
2. ضع كلمة مرور من القائمة الخاصة
3. ✅ سيتم تحديده كأدمن تلقائياً

### 🎨 **في شاشة التسجيل:**

```dart
// استخدام Widget الجديد
AdminRegistrationWidget(
  onAdminCodeChanged: (adminCode) {
    // حفظ الكود للاستخدام في التسجيل
    _adminSecretCode = adminCode;
  },
)

// في دالة التسجيل
await authService.register(
  user,
  password,
  adminSecretCode: _adminSecretCode, // تمرير الكود
);
```

### 🔍 **فحص النتيجة:**

#### تحقق من النوع بعد التسجيل:
```dart
final user = await authService.getCurrentUser();
if (user?.type == 'admin') {
  print('✅ تم التسجيل كأدمن بنجاح!');
} else {
  print('ℹ️ تم التسجيل كمستخدم عادي');
}
```

### 📋 **أمثلة عملية:**

#### مثال 1: تسجيل بإيميل أدمن
```
Email: admin@elltall.com
Password: mypassword123
النتيجة: ✅ أدمن تلقائياً
```

#### مثال 2: تسجيل بكود سري
```
Email: user@gmail.com
Password: normalpassword
Admin Code: ADMIN_SECRET_2024
النتيجة: ✅ أدمن بالكود السري
```

#### مثال 3: تسجيل بكلمة مرور خاصة
```
Email: anyone@example.com
Password: ADMIN_MASTER_2024
النتيجة: ✅ أدمن بكلمة المرور الخاصة
```

#### مثال 4: تسجيل عادي
```
Email: user@gmail.com
Password: normalpassword
النتيجة: ℹ️ مستخدم عادي
```

### 🛡 **الأمان:**

1. **كلمات المرور الخاصة**: تُحفظ في الكود فقط
2. **الكود السري**: يمكن تغييره بسهولة
3. **قائمة الإيميلات**: محددة مسبقاً ومحمية
4. **التشفير**: كل البيانات محمية في قاعدة البيانات

### 🚀 **التطبيق العملي:**

#### لتجربة النظام الآن:
1. شغّل التطبيق
2. اذهب لشاشة التسجيل
3. جرب أي من الطرق المذكورة
4. تحقق من النتيجة في قاعدة البيانات
