## 🔄 إضافة فئة جديدة | Adding New Category

```dart
// 1. إضافة التكوين في _allConfigs()
static List<CategoryFieldConfig> _allConfigs() {
  return [
    // ... فئات موجودة
    _newCategoryConfig(),
  ];
}

// 2. إنشاء ميثود التكوين
static CategoryFieldConfig _newCategoryConfig() {
  return CategoryFieldConfig(
    categoryId: 'new_category',
    categoryName: 'New Category',
    fields: [
      DynamicField(
        id: 'field_id',
        label: 'اسم الحقل',
        type: FieldType.singleChoice,
        isRequired: true,
        options: [
          FieldOption(id: 'opt1', label: 'خيار 1', price: 0),
          FieldOption(id: 'opt2', label: 'خيار 2', price: 5),
        ],
      ),
    ],
  );
}
```

## 📝 ملاحظات مهمة | Important Notes

1. **التطابق بالاسم:** النظام يطابق اسم الفئة (case-insensitive)
2. **الحقول الشرطية:** بعض الحقول تظهر فقط عند اختيار قيمة معينة
3. **الأسعار الإضافية:** كل خيار يمكن أن يضيف سعراً للمنتج
4. **الحقول المطلوبة:** يجب ملء الحقول المطلوبة قبل الحفظ
5. **قابلية التوسع:** يمكن إضافة فئات جديدة بسهولة
